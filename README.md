# 🎌 Stitch Anime Motion Portal

A bold and modern anime web portal featuring SHONEN_PULSE - a dynamic web application for anime enthusiasts.

## ✨ Features

- **Modern Dark UI** - Sleek dark theme with vibrant accent colors
- **Responsive Design** - Mobile-friendly layout using Tailwind CSS
- **Material Design Icons** - Professional icons for enhanced UX
- **Multiple Pages** - Navigate through different sections of the portal
- **Custom Typography** - Space Grotesk and Be Vietnam Pro fonts

## 📁 Project Structure

```
stitch_anime_motion_portal/
├── code.html           # Main homepage
├── Page 2.html         # Secondary page
├── Page 3.html         # Additional content page
├── Page 4.html         # Final section
└── README.md           # Project documentation
```

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework
- **Google Fonts** - Space Grotesk, Be Vietnam Pro
- **Material Symbols** - Icon library

## 🚀 Getting Started

1. Open any `.html` file in your web browser
2. Navigate between pages using the provided navigation
3. Enjoy the anime portal experience!

## 📱 Responsive Design

The site is optimized for all screen sizes with `viewport` meta tag and Tailwind's responsive utilities.
